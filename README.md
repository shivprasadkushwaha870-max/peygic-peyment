# PayGic Payment Gateway

Standalone payment page for PayGic UPI payments.

## Live URL
https://peygic-payment.vercel.app

## API Endpoints
- `/api/token` - Generate PayGic API token
- `/api/payment` - Create payment page
- `/api/status` - Check payment status
- `/api/callback` - Payment callback handler
